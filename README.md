# Furni - Modern Multi-Page E-Commerce Furniture Website

Furni is a premium, multi-page e-commerce landing page and store built for modern interior design and furniture shops. This project demonstrates clean UI architecture, multi-page navigation, and custom smooth transitions, ensuring an elegant user experience.

## 🚀 Live Demo
[Click here to view the live website](https://basmala-taima.github.io/Furni/)

## ✨ Features
* **Multi-Page Architecture:** Includes fully styled sub-pages (Shop, About Us, Services, Blog, Contact Us, Cart, and Checkout) with smooth routing and seamless connectivity.
* **Responsive Layouts:** Built using Bootstrap’s grid system and components, optimized for flawless viewing across mobile, tablet, and desktop screens.
* **Custom CSS Transitions & Animations:** Enhanced UI interactiveness using advanced pure CSS for hover effects, smooth transitions, and custom component styling beyond framework defaults.
* **E-Commerce Flow Elements:** Formatted cart overview, interactive billing detail forms, coupon code section, and a custom order placement confirmation page.

## 🛠️ Tech Stack
* **HTML5:** For semantic web structuring across multiple pages.
* **Bootstrap 5:** For fast layout setup, navigation components, and responsive grid alignment.
* **CSS3 (Custom):** For bespoke animations, color consistency, transitions, and layout overrides.
* **JavaScript (Vanilla):** For basic interactive elements and behavior.
